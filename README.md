# ymorzincupfinal.com

This repository contains the source code for [ymorzincupfinal.com](https://ymorzincupfinal.com).

## Local Development

1. Install dependencies: `pnpm install`

2. Run development server, this will reload automatically when you make changes to the source files: `pnpm dev`

3. Build a production build to make sure it looks the same: `pnpm build && pnpm preview`
